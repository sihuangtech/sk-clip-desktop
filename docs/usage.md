# 使用指南

本文档将指导您如何使用彩旗剪辑的主要功能。

## 应用概览

彩旗剪辑是一个集成了视频翻译、文档导入、视频剪辑和语音合成功能的跨平台桌面应用。它旨在帮助您方便地将文档内容和视频素材整合成带有翻译和配音的视频，特别适用于制作演示、教程和汇报视频。

有关应用的功能特点和技术栈的更多信息，请参考 [README](../README.md) 文件。

## 核心工作流程

以下是应用的完整工作流程：

1. **准备素材**：收集您需要使用的文档（PPTX, MD, PDF）和视频文件。
2. **导入内容**：将文档文件导入应用，应用将解析并准备用于视频的内容。
3. **插入视频**：导入您的视频素材，并将其添加到时间线中。
4. **编辑与组合**：在时间线上编排文档内容和视频片段的顺序和时长，进行基础剪辑。
5. **添加语音**：
    * **视频翻译配音/字幕**：选择视频片段进行翻译，应用将生成目标语言的配音和字幕，并自动同步。
    * **文本转语音 (TTS)**：输入或粘贴文本，选择语言和音色，生成配音并插入到时间线中。
6. **导出视频**：将最终组合好的内容导出为完整的视频文件。

## 已实现功能

### 视频处理 (FFmpeg)

应用已集成 FFmpeg，支持以下视频处理功能：

1. **音频提取**：从视频中提取音频轨道，支持 MP3、WAV、AAC、OGG、FLAC 等格式。
2. **视频裁剪**：按照时间戳裁剪视频片段。
3. **视频合并**：将多个视频文件合并为一个。
4. **字幕烧录**：将字幕文件嵌入视频中。
5. **视频缩放**：调整视频分辨率。
6. **缩略图生成**：从视频中提取指定时间点的缩略图。
7. **视频信息获取**：获取视频的元数据信息（时长、分辨率、编码格式等）。

### 时间线项目管理

应用已实现可视化时间线编辑器，支持以下功能：

1. **项目创建**：创建新的时间线项目，自动添加视频和音频轨道。
2. **项目保存与加载**：保存项目文件，随时继续编辑。
3. **视频导出**：使用 FFmpeg 将时间线内容导出为视频文件，支持自定义分辨率、质量和格式。
4. **项目管理**：列出、删除已保存的项目。

### 视频翻译

应用支持视频翻译流程：

1. 启动应用。
2. 在主界面找到文件选择区域，点击上传一个视频文件。
3. 选择视频的源语言和您希望翻译成的目标语言（支持中文、英文、日语、韩语）。
4. 点击"开始翻译"按钮。
5. 翻译完成后，查看输出文件路径。
