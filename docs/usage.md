# 使用指南

本文档将指导您如何使用视频创作与翻译工具的主要功能。

## 应用概览

视频创作与翻译工具是一个集成了视频翻译、文档导入、视频插入和语音合成功能的桌面应用。它旨在帮助您方便地将文档内容和视频素材整合成带有翻译和配音的视频，特别适用于制作演示、教程和汇报视频。

有关应用的功能特点和技术栈的更多信息，请参考 [README](../README.md) 文件。

## 核心工作流程 (规划中的完整流程)

以下是应用规划中的完整工作流程，当前版本实现了部分基础交互：

1.  **准备素材**：收集您需要使用的文档（PPTX, MD, PDF）和视频文件。
2.  **导入内容**：将文档文件导入应用，应用将解析并准备用于视频的内容。
3.  **插入视频**：导入您的视频素材，并将其添加到时间线中。
4.  **编辑与组合**：在时间线上编排文档内容和视频片段的顺序和时长，进行基础剪辑。
5.  **添加语音**：
    *   **视频翻译配音/字幕**：选择视频片段进行翻译，应用将生成目标语言的配音和字幕，并自动同步。*（未来计划使用离线的免费开源项目实现此功能，具体库待定）*
    *   **文本转语音 (TTS)**：输入或粘贴文本，选择语言和音色，生成配音并插入到时间线中。*（未来计划使用离线的免费开源项目实现此功能，具体库待定）*
6.  **导出视频**：将最终组合好的内容导出为完整的视频文件。

## 当前版本功能使用 (原型演示)

当前版本主要提供了基础的应用框架和模拟的视频翻译流程，帮助您了解应用的基本交互。

### 视频翻译 (模拟实现)

1.  启动应用。
2.  在主界面找到文件选择区域，点击上传一个视频文件。
3.  选择视频的源语言和您希望翻译成的目标语言（目前支持中文、英文、日语、韩语）。
4.  点击"开始翻译"按钮。
5.  应用会进入"正在翻译"状态（此过程为模拟耗时）。
6.  翻译完成后，应用会显示"翻译完成！"，并在界面上播放模拟的翻译结果视频。
7.  您可以查看输出文件的路径。

**请注意**：当前版本的视频翻译功能仅为流程演示，没有实际集成语音识别、翻译、语音合成或视频处理功能。播放的视频是简单复制的输入文件，不包含实际的翻译效果。

### 文档导入、视频插入、语音合成 (待实现)

这些功能在当前版本中尚未实现，但已在应用设计和 README 中进行了规划。它们将是未来版本的重点开发内容。敬请期待后续更新。
