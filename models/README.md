# AI 模型存储目录

这个目录用于存储彩旗剪辑应用所需的各种AI模型文件。

## 目录结构

```
models/
├── whisper/          # Whisper 语音识别模型
│   ├── base/         # 基础模型 (74MB)
│   ├── small/        # 小型模型 (244MB)
│   ├── medium/       # 中型模型 (769MB)
│   └── large/        # 大型模型 (1550MB)
├── translation/      # 文本翻译模型
│   ├── opus-mt/      # OPUS-MT 翻译模型系列
│   └── m2m100/       # Facebook M2M100 多语言翻译模型
└── tts/             # 文本转语音模型
    ├── espeak/       # eSpeak TTS 引擎
    ├── festival/     # Festival TTS 系统
    └── neural/       # 神经网络 TTS 模型
```

## 模型下载说明

### Whisper 模型
- **base**: 适合快速处理，准确度中等
- **small**: 平衡速度和准确度
- **medium**: 较高准确度，处理速度适中
- **large**: 最高准确度，处理速度较慢

### 翻译模型
- 支持中文、英文、日文、韩文、西班牙文、俄文、德文、法文、葡萄牙文等语言
- 模型会根据需要自动下载

### TTS 模型
- 支持多种语言的语音合成
- 包含男声和女声选项

## 使用说明

1. 应用首次启动时会检查必要的模型是否存在
2. 如果模型不存在，会自动从官方源下载
3. 下载的模型会缓存在对应的子目录中
4. 用户可以在设置中选择使用的模型类型

## 注意事项

- 模型文件较大，请确保有足够的磁盘空间
- 首次下载可能需要较长时间，请保持网络连接稳定
- 模型文件不应手动删除，除非需要重新下载
- 建议将此目录添加到杀毒软件的白名单中
